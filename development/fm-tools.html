Internal Tools for FileMaker

2018 – 2019

While contracting at FileMaker, I coded a handful of internal tools built mainly in JavaScript & JavaScript-based libraries. Some of the tools include:

  • a bookmarklet that gets HTML information on hover, compares it to hosted JSON data, and displays a status report
  • a Grunt script that compares HTML content with JSON data and outputs new HTML files with the correct content if errors are found
  • a Grunt script that writes CSS files for images based on their dimensions

These projects involved light research into which technologies to use and independent learning.
